NAME

i.landsat8.swlst.py - Practical split-window algorithm estimating Land Surface Temperature from Landsat 8 OLI/TIRS imagery (Du, Chen; Ren, Huazhong; Qin, Qiming; Meng, Jinjie; Zhao, Shaohua. 2015)

KEYWORDS

imagery, split window, land surface temperature, lst, landsat8

SYNOPSIS

i.landsat8.swlst.py
i.landsat8.swlst.py --help
i.landsat8.swlst.py [-ikc] [prefix=prefix string] b4=Band 4 b5=Band 5 [b10=Band 10] [b11=Band 11] t10=Temperature (10) t11=Temperature (11) qab=QA band [clouds=cloud masking] [e10=Emissivity B10] [e11=Emissivity B11] [landcover=land cover map name] emissivity_class=emissivity class lst=lst output [cwv=cwv output] [--overwrite] [--help] [--verbose] [--quiet] [--ui]

Flags:

-i
Print out model equations
-k
Keep current computational region settings
-c
Apply Celsius colortable to output LST map
--overwrite
Allow output files to overwrite existing files
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

prefix=prefix string
OLI band names prefix
Prefix of Landsat8 OLI band names
Default: B
b4=Band 4 [required]
Band 4 - Red (0.64 - 0.67 microns)
b5=Band 5 [required]
Band 5 - Near Infra-Red (0.85 - 0.88 microns)
b10=Band 10
Band 10 - TIRS (10.60 - 11.19 microns)
b11=Band 11
Band 11 - TIRS (11.50 - 12.51 microns)
t10=Temperature (10) [required]
Brightness temperature (K) from band 10
t11=Temperature (11) [required]
Brightness temperature (K) from band 11
qab=QA band [required]
Landsat 8 quality assessment band
clouds=cloud masking
Confidence level for cloud masking (see...) -- ToDo!
Options: clouds, cirrus, both
Default: high
e10=Emissivity B10
Emissivity for Landsat 8 band 10
e11=Emissivity B11
Emissivity for Landsat 8 band 11
landcover=land cover map name
Land cover map
emissivity_class=emissivity class [required]
Manual selection of land cover class to retrieve average emissivity from a look-up table (case sensitive)
Options: Cropland, Forest, Grasslands, Shrublands, Wetlands, Waterbodies, Tundra, Impervious, Barren, Snow
lst=lst output [required]
Name for output Land Surface Temperature map
Default: lst
cwv=cwv output
Name for output Column Water Vapor map [optional]
